# Employee Model

A simple Java `Employee` class used to store and manage employee details such as ID, name, position, and salary.

## Features
- Stores employee information
- Uses getters and setters
- Displays employee details

## Technologies Used
- Java
- OOP concepts (Encapsulation)

## How to Use
Create an `Employee` object by providing the ID, name, position, and salary, then use the methods to view or update the employee details.
